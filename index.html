Mengapa Ini Bisa Terjadi? (Cara Kerja SW)

Edukasi bagaimana Service Worker mampu memotong batasan tab tertutup.

Mekanisme `event.waitUntil()`

Ketika tab ditutup, browser biasanya akan menghentikan seluruh proses JavaScript. Namun, Service Worker menggunakan API event.waitUntil() yang secara khusus meminta sistem operasi menahan proses tetap berjalan hingga *Promise* selesai dikerjakan.

Pembatasan Suara (Fakta Mendasar)

Meskipun notifikasi berhasil terpicu dari Service Worker saat tab mati, Anda **tetap tidak akan mendengar suara kustom** (seperti suara "Uang Masuk"). Anda hanya akan mendengar suara bawaan pabrik sistem operasi HP karena Service Worker tidak diizinkan memanggil *Audio Synth* browser saat tab tertutup.

Metode Penjadwalan Tab Ditutup? Hasil Suara
JS Standard (setTimeout) Mati total Tidak berbunyi
Service Worker Thread Tetap Menyala Hanya Nada OS Bawaan
Flutter Native (FCM) Tetap Menyala Suara Kustom MP3
🎯 Fakta Terbukti: Melalui modifikasi ini, kita berhasil memicu notifikasi saat tab ditutup memakai Service Worker asli tanpa server eksternal, tetapi pembatasan audio kustom browser tetap berlaku 100%!
