Kenapa Suara Kustom Tidak Bunyi?

Properti sound di NotificationOptions sudah ditiadakan (deprecated) oleh Google Chrome, Safari, dan Firefox sejak lama. Browser secara sepihak memaksakan suara sistem bawaan OS demi mencegah pelecehan audio (spam suara iklan).

Aturan Autoplay Kebijakan Browser

Suara dari halaman web (menggunakan Web Audio API / tag HTML5 Audio) Wajib dipicu dari dalam tab aktif setelah ada interaksi fisik pengguna (seperti klik). Jika tab ditutup atau disembunyikan lama, browser akan langsung mencekal .play().

Platform / Mode Bisa Notif? Suara Kustom?

Web App (Tab Ditutup) YA (Lewat SW) TIDAK BISA
